    Homework 1: Food Icons

    Submit Assignment
    • Due Tuesday by 11:59pm
    • Points 3
    • Submitting a file upload
    • File Types ai, jpg, and png
    • Available after Sep 11 at 8:30pm

    Using what you learned from Lesson 1 and Tutorial 1, create three icons of your favorite foods! Try to base it off the tutorial we did in class where we started off with basic shapes, and then used Pathfinder to combine them to create more complex ones. 

    Use SHIFT-O to create new artboards in your workspace for multiple icons. To submit, go to File > Export As and check off “PNG” and “Use Artboards”.

    Make sure to submit your homework as both Ai and PNG/JPEG files! Please do not submit homework as any other file type except Ai or PNG/JPEG.
